# Panopricon API

## 起動コマンド

```bash
# 開発環境起動
$ make
# linter起動
$ make lint
# formater起動
$ make format
# test実行
$ make test
# test watch
$ make watch
```

# vscode 起動

```bash
$ poetry shell
$ code .
```
