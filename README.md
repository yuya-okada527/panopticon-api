# Panopticon API

セルフモニタリングツール「Panopticon」のコア機能の API。

## 利用ツール

- 言語: Ruby3.0.2
- フレームワーク: Ruby on Rails
- データベース: MySQL
- DB スキーマ管理: ridgepole
- JSON シリアライザ: jb
- HTTP クライアント: faraday
- テストフレームワーク: RSpec
- シード作成: factory-bot
- ドキュメント: OpenAPI

## 開発環境

- API: http://localhost:3000
- DB クライアント: http://localhost:8888
